# Inquiry Matrix

> 提问决定成败，而非答案。—— Shensist Art Intelligence Studio

Inquiry Matrix 是一款采用 C端免费开源理念 的思维辅助工具。通过极简的设计与先进的交互理念，辅助用户将零散的点子进行第一性原理级的剖析与落地。

## 工程结构
- `/src` : 包含核心逻辑脚本 `app.js` (ESM)。
- `/public` : 包含公共静态资产，例如 Logo 等。
- `index.html` : 极简无依赖的前端 Web 页面。
